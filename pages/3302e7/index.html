Chapter 1——向量空间

# 1 定义

  • 数域定义 数域FFF，是至少包含000和111的数集，并满足以下性质：

    1. ∀a∈F,−a∈F\forall a\in F, -a\in F∀a∈F,−a∈F
    2. ∀b∈F(b≠0),b−1∈F\forall b\in F(b\neq 0), b^{-1}\in F∀b∈F(b=0),b−1∈F
    3. ∀a,b∈F,a+b∈F\forall a, b\in F, a+b\in F∀a,b∈F,a+b∈F
    4. ∀a,b∈F,a⋅b∈F\forall a,b\in F, a\cdot b\in F∀a,b∈F,a⋅b∈F

    矩阵论中最常用到的两个数域是RRR（实数域）和CCC（复数域）

  • 代数系统定义

    代数系统通常是定义了一些运算和运算规则的集合。描述一个代数系统需要：

    1. 一组元素
    2. 运算
    3. 运算规则

  • 几何向量定义

    有大小有方向的量，可以用有向线段表示，如α⃗\vec{\alpha}α。有加法和数乘运算。

  • 向量空间定义

    一个域 FFF（底域）上的向量空间（线性空间或线性向量空间）VVV 是一组元素（称为向量）以及加法和标量乘法这两种运算，并满足以下条件：

    1. 闭包性

      • ∀x,y∈V,x+y∈V\forall x,y\in V, x+y\in V∀x,y∈V,x+y∈V，且x+yx+yx+y运算结果唯一

      • ∀a∈F,x∈V,a⋅x∈V\forall a\in F,x\in V, a\cdot x\in V∀a∈F,x∈V,a⋅x∈V，且a⋅xa\cdot xa⋅x运算结果唯一

    2. 加法公理

      • 交换律：∀x,y∈V,x+y=y+x\forall x,y\in V, x+y=y+x∀x,y∈V,x+y=y+x
      • 结合律：∀x,y,z∈V,x+(y+z)=(x+y)+z\forall x,y,z\in V, x+(y+z)=(x+y)+z∀x,y,z∈V,x+(y+z)=(x+y)+z
      • 存在零向量：∀x∈V,x+0=0+x=x\forall x\in V,x+0=0+x=x∀x∈V,x+0=0+x=x
      • 存在相反向量：∀x∈V,∃(−x)∈V,x+(−x)=0\forall x\in V,\exist (-x)\in V,x+(-x)=0∀x∈V,∃(−x)∈V,x+(−x)=0

    3. 标量乘法公理

      • 结合律：∀a,b∈F,x∈V,a⋅(b⋅x)=(ab)⋅x\forall a,b\in F, x\in V,a\cdot (b\cdot x)=(ab)\cdot x∀a,b∈F,x∈V,a⋅(b⋅x)=(ab)⋅x
      • 分配律111：∀a∈F,x,y∈V,a⋅(x+y)=a⋅x+a⋅y\forall a\in F, x,y\in V,a\cdot(x+y)=a\cdot x+a\cdot y∀a∈F,x,y∈V,a⋅(x+y)=a⋅x+a⋅y
      • 分配律222：∀a,b∈F,x∈V,(a+b)⋅x=a⋅x+b⋅x\forall a,b\in F, x\in V,(a+b)\cdot x=a\cdot x + b\cdot x∀a,b∈F,x∈V,(a+b)⋅x=a⋅x+b⋅x
      • 存在单位元素：∀x∈V,1⋅x=x\forall x\in V,1\cdot x=x∀x∈V,1⋅x=x

  • 向量空间重点

    定义一个向量空间需要：一个集合VVV，一个数域FFF，两种运算，八种运算规则。

  • 常见向量空间

    1. Rm×n,(Rm×1=Rm)R^{m\times n},(R^{m\times 1}=R^m)Rm×n,(Rm×1=Rm)：m×nm\times nm×n的实数矩阵集合，在RRR上的向量空间
    2. Cm×n,(Cm×1=Rm)C^{m\times n},(C^{m\times 1}=R^m)Cm×n,(Cm×1=Rm)：m×nm\times nm×n的复数矩阵集合，在CCC上的向量空间
    3. C[a,b]C_{[a,b]}C[a,b]​：闭区间上的连续函数集合，在RRR上的向量空间
    4. PnP_nPn​：次数小于 nnn 的实多项式的集合，在RRR上的向量空间

# 2 子空间

  • 子空间定义

    如果SSS是向量空间VVV在数域FFF上的一个非空子集，且满足闭包性，则SSS就是VVV的一个子空间。

    由VVV的零向量所组成的自己{0}\{0\}{0}是VVV的一个子空间，称为零子空间，向量空间VVV本身也是VVV的一个子空间，它们都称为VVV的平凡子空间，VVV的其他子空间称为非平凡子空间。

  • 子空间重点

    VVV 的子空间 SSS 以及 VVV 的加法和标量乘法运算满足向量空间定义中的所有条件。因此，向量空间的每个子空间本身就是一个向量空间。SSS 的底层域与 VVV 的底层域相同。

  • 零空间定义

    设A∈Fm×n,N(A)={x∈Fn∣Ax=0}A\in F^{m\times n},N(A)=\{x\in F^n|Ax=0\}A∈Fm×n,N(A)={x∈Fn∣Ax=0}，则N(A)N(A)N(A)为FnF^nFn的子空间，N(A)N(A)N(A)称为AAA的零空间。

  • 向量的线性相关性

    设VVV是数域FFF上的线性空间，a1,⋯,an∈F,v1,⋯,vn∈Va_1,\cdots,a_n\in F,v_1,\cdots,v_n\in Va1​,⋯,an​∈F,v1​,⋯,vn​∈V，则a1v1+⋯+anvna_1v_1+\cdots+a_nv_na1​v1​+⋯+an​vn​就是v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​的线性组合。

    若存在nnn个不全为零的数a1,⋯,an∈Fa_1,\cdots,a_n\in Fa1​,⋯,an​∈F，使得a1v1+⋯+anvn=0a_1v_1+\cdots+a_nv_n=0a1​v1​+⋯+an​vn​=0，则称v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​线性相关，否则就称为线性无关。

    线性相关的充要条件是其中有一个向量是其余向量的线性组合

  • 生成集定义

    span(K)={v∣v是向量K的一个线性组合}span(K)=\{v|v\texttt{是向量}K\texttt{的一个线性组合}\}span(K)={v∣v是向量K的一个线性组合}，即为向量KKK的线性组合生成的集合。如果KKK是向量空间VVV中的有限集，那么span(K)span(K)span(K)也就是VVV的子空间。

    如果v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​是向量空间VVV的向量，且V=span{v1,⋯,vn}V=span\{v_1,\cdots,v_n\}V=span{v1​,⋯,vn​}，则集合{v1,⋯,vn}\{v_1,\cdots,v_n\}{v1​,⋯,vn​}称为VVV的生成集。

  • 子空间的交集、和

    1. 设U1,U2U_1,U_2U1​,U2​为向量空间VVV的子空间，则U1∩U2={v∣v∈U1,v∈U2}U_1\cap U_2=\{v|v\in U_1, v\in U_2\}U1​∩U2​={v∣v∈U1​,v∈U2​}，U1∩U2U_1\cap U_2U1​∩U2​也是VVV的子空间。

    2. 设U1,U2U_1,U_2U1​,U2​为向量空间VVV的子空间，则U1+U2={v1+v2∣v1∈U1,v2∈U2}U_1+U_2=\{v_1+v_2|v_1\in U_1,v_2\in U_2\}U1​+U2​={v1​+v2​∣v1​∈U1​,v2​∈U2​}，U1+U2U_1+ U_2U1​+U2​也是VVV的子空间。

      如果U1=span(u1,⋯,uk),U2=span(w1,⋯,ws)U_1=span(u_1,\cdots_,u_k),U_2=span(w_1,\cdots,w_s)U1​=span(u1​,⋯,​uk​),U2​=span(w1​,⋯,ws​)，则U1+U2=span(u1,⋯,uk,w1,⋯,ws)U_1+U_2=span(u_1,\cdots,u_k,w_1,\cdots,w_s)U1​+U2​=span(u1​,⋯,uk​,w1​,⋯,ws​)。

  • 子空间的直和

    设V1,V2V_1,V_2V1​,V2​是向量空间VVV的两个子空间，如果和V1+V2V_1+V_2V1​+V2​中每一个向量α\alphaα可唯一表示成α=α1+α2,α1∈V1,α2∈V2\alpha=\alpha_1+\alpha_2,\alpha_1\in V_1,\alpha_2\in V_2α=α1​+α2​,α1​∈V1​,α2​∈V2​，则称和V1+V2V_1+V_2V1​+V2​为直和，记为V1+V2V_1+V_2V1​+V2​。

    和V1+V2V_1+V_2V1​+V2​是直和⟺\Longleftrightarrow⟺和V1+V2V_1+V_2V1​+V2​中零向量的表示法唯一，即若α1+α2=0(α1∈V1,α2∈V2)\alpha_1+\alpha_2=0(\alpha_1\in V_1,\alpha_2\in V_2)α1​+α2​=0(α1​∈V1​,α2​∈V2​)，则α1=0,α2=0⟺V1∩V2={0}⟺dim⁡(V1+V2)=dim⁡(V1)+dim⁡(V2)\alpha_1=0,\alpha_2=0\Longleftrightarrow V_1\cap V_2=\{0\}\Longleftrightarrow\dim(V_1+V_2)=\dim(V_1)+\dim(V_2)α1​=0,α2​=0⟺V1​∩V2​={0}⟺dim(V1​+V2​)=dim(V1​)+dim(V2​)

  • 补空间定义

    如果$$V=V_1\oplus V_2$$，
    我们则称V1V_1V1​和V2V_2V2​互为补空间，即V1V_1V1​是V2V_2V2​的补。

  • 补空间定理 如果UUU是VVV的子空间，则存在VVV的子空间WWW，使得V=U⊕WV=U\oplus WV=U⊕W

# 3 基、坐标和维数

  • 基、坐标定义

    nnn个向量v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​是向量空间VVV的一组基，当且仅当：

    1. v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​线性无关
    2. V=span(v1,⋯,vn)V=span(v_1,\cdots,v_n)V=span(v1​,⋯,vn​)

    基不是唯一的，但VVV的所有基中的向量个数是相同的

    设aaa是VVV中的任一向量，则aaa可以唯一的表示为基v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​的线性组合a=k1v1+⋯+knvna=k_1v_1+\cdots+k_nv_na=k1​v1​+⋯+kn​vn​，其中系数k1,⋯,knk_1,\cdots,k_nk1​,⋯,kn​称为aaa在基v1,⋯,vnv_1,\cdots,v_nv1​,⋯,vn​下的坐标，记为(k1,⋯,kn)T(k_1,\cdots,k_n)^T(k1​,⋯,kn​)T。

  • 维数定义

    如果向量空间VVV的基由nnn个向量组成，则我们称VVV的维数是nnn。

  • 定理

    在nnn维线性空间VVV中，任意一个线性无关的向量组a1,⋯,ara_1,\cdots,a_ra1​,⋯,ar​都可以扩充为VVV的一组基。

  • 维数公式

    设U1,U2U_1,U_2U1​,U2​为向量空间VVV的两个子空间，则

    dim⁡(U1+U2)=dim⁡(U1)+dim⁡(U2)−dim⁡(U1∩U2)\dim(U_1+U_2)=\dim(U_1)+\dim(U_2)-\dim(U_1\cap U_2)dim(U1​+U2​)=dim(U1​)+dim(U2​)−dim(U1​∩U2​)

# 5 行空间和列空间

  • 矩阵的秩

    如果矩阵AAA的秩为rrr，则说明：

    1. 存在一个r×rr×rr×r子矩阵，其行列式不为零；和
    2. 所有的 (r+1)×(r+1)(r+1)\times (r+1)(r+1)×(r+1) 的子矩阵的行列式为零。

  • 行空间和列空间

    设A∈Fm×nA\in F^{m\times n}A∈Fm×n

    1. 行空间：由AAA的行向量生成的F1×nF^{1\times n}F1×n的子空间。（也为ATA^TAT的列空间）
    2. 列空间：由AAA的列向量生成的Fm×1F^{m\times 1}Fm×1的子空间。（也为ATA^TAT的行空间）

  • 行等价条件

    矩阵AAA和BBB被称为是行等价⟺\Longleftrightarrow⟺BBB可以由AAA进行初等行变换得到。

    特别地，对于非奇异矩阵，有充要条件是存在一个非奇异矩阵MMM使得，MA=BMA=BMA=B。

  • 行等价性质

    设矩阵A,BA,BA,B是两个行等价的矩阵，则：

    1. 它们有相同的行空间。
    2. 如果AAA中的列向量ai1,⋯,aika_{i_1},\cdots,a_{i_k}ai1​​,⋯,aik​​是线性无关的，则BBB中的列向量bi1,⋯,bikb_{i_1},\cdots,b_{i_k}bi1​​,⋯,bik​​也是线性无关的

  • 列空间性质

    1. 线性系统Ax=bAx=bAx=b相容（有解）⟺\Longleftrightarrow⟺bbb在AAA的列空间里
    2. Ax=bAx=bAx=b相容当且仅当rank(A)=rank(A,b)rank(A)=rank(A,b)rank(A)=rank(A,b)，即等价于AAA的列空间等于(A,b)(A,b)(A,b)的列空间
    3. 如果∀b∈Fm\forall b\in F^m∀b∈Fm，Ax=bAx=bAx=b相容，说明AAA的列空间是FmF^mFm。
    4. 如果∀b,Ax=b\forall b,Ax=b∀b,Ax=b至多只有一个解，说明AAA的列向量是线性无关的，则AAA的列向量是AAA的列空间的基，等价于AAA是非奇异矩阵。

  • 秩——零度定理

    设AAA为m×nm\times nm×n矩阵，则rank(A)+rank(N(A))=0rank(A)+rank(N(A))=0rank(A)+rank(N(A))=0

  • 秩和维数

    设AAA是m×nm\times nm×n矩阵，AAA的行空间维数等于AAA的列空间维数，即dim⁡(R(AT))=dim⁡(R(A))\dim(R(A^T))=\dim(R(A))dim(R(AT))=dim(R(A))，其中R(AT)R(A^T)R(AT)表示ATA^TAT的列空间，即AAA的行空间。

    虽然矩阵AAA的行空间和列空间不相同，但是它们有相同的维数，都为AAA的秩，矩阵AAA在初等变换下秩是不变的。
